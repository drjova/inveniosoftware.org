1
Clone and Create Branch

Download the website code and create a new git branch like this:

$ git clone git@github.com:inveniosoftware/inveniosoftware.org.git $ cd inveniosoftware.org $ git checkout -b new-site

2
Beatiful Screenshot

Take the best screenshot you can for your site and place it in the right folder. Name the image after your site, as in the following example:

$ mv mysite.png base/static/img/screenshots/

3
JSON file with your site's data

Create a JSON file containing your site's data as in the example:

$ echo '{ "name": "CERN Document Server", "description": "CERN official repository for publications, articles, reports and multimedia content in HEP.", "url": "http://cds.cern.ch", "screenshot_filename": "mysite.png", "types": ["research-data", "library", "multimedia", "institutional-repository"] }' > base/sites/mysite.json

4
Back to GitHub

Now you are all done, you only need to commit your changes, push them to your repo and do a pull request so we can integrate it!

You will need first to do:

$ git add base/static/img/screenshots/mysite.png $ git add base/sites/mysite.json $ git commit -a -m "added my site foo to showcase" $ git push origin new-site

Once this is done, go to your github repo and do a Pull Request.
